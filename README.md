## 🚀 Setup rápido

```bash
cd trip-conecta-panel
npm install
cp .env.example .env.local
# Editar .env.local con tus valores
npm run dev
```

Panel local: `http://localhost:3000`

## 🧪 Verificación antes de commit

```bash
npm run build
```

## 🚀 Deploy

- Rama: `main`
- Deploy: automático en Coolify al hacer push

⚠️ **No hacer push sin verificar que `npm run build` pasa localmente.**
